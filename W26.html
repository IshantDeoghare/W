PICT B.E. Structure

First Year (FE)

  • Engineering Mathematics I: Calculus, sequences & series.
  • Engineering Physics: Mechanics, waves, optics.
  • Basic Electrical Engg.: Circuits, DC/AC fundamentals.
  • Programming for Problem Solving: C programming basics.
  • Workshop Practice: Hands-on fabrication skills.

Second Year (SE)

  • Data Structures: Arrays, linked lists, trees.
  • Digital Electronics: Logic gates, flip-flops.
  • Computer Organization: CPU architecture, memory.
  • Object Oriented Programming: Java/C++ fundamentals.
  • Discrete Mathematics: Graphs, combinatorics.

Third Year (TE)

  • Operating Systems: Processes, scheduling.
  • Database Management: SQL, normalization.
  • Computer Networks: TCP/IP, routing.
  • Principles of Communication: Signals, modulation.
  • Software Engineering: Lifecycle models, UML.

Final Year (BE)

  • Machine Learning: Supervised & unsupervised methods.
  • Cloud Computing: IaaS, PaaS, SaaS concepts.
  • Cyber Security: Threat models, cryptography.
  • Elective Project: Industry-oriented capstone project.
  • Professional Ethics: Ethical practices in engineering.
